# Design System (Strict)

## Color Distribution
- 70% White (#FFFFFF)
- 20% Cyan (#06B6D4)
- 10% Black (#000000)

## Rules
- White background
- Cyan primary buttons
- Black text
- Cards:
    - white background
    - subtle cyan border
    - soft shadow
- Hover:
    scale(1.02)
    transition 0.2s ease
- Step transitions:
    small fade + slight slide using framer-motion

## DO NOT
- Use gradients
- Use dark theme
- Use glassmorphism
- Use AI-style layouts
- Over-animate

Clean. Structured. Minimal.
