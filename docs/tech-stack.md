# Tech Stack (Non-Negotiable)

## Frontend
- React
- TypeScript (strict mode)
- Vite
- shadcn/ui
- TailwindCSS
- React Router
- Axios
- React Hook Form
- Zod
- Zustand (or Context API)
- Framer Motion (minimal use only)

## Backend
- Python
- FastAPI
- SQLAlchemy ORM
- Pydantic
- PostgreSQL

## Testing
- Vitest (frontend)
- React Testing Library
- Pytest (backend)
- Playwright (E2E)

## Containerization
- Docker
- docker-compose

## DO NOT
- Change stack
- Introduce Redux
- Introduce heavy styling frameworks
- Replace shadcn
- Add unnecessary UI libraries
- Overengineer architecture
