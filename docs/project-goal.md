# Project Goal

User builds a connected multi-destination trip across European capital cities.

## Flow

1. Onboarding
2. Select European capital cities
3. Choose ONE global vacation start date
4. Define number of days per destination
5. Generate connected flight route (mock engine)
6. Select flights
7. Select hotels
8. Select attractions
9. View summary
10. Confirm with button: "Fly Me A Travel"

Flights are mock data only.
